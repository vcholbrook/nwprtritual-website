# nwprtritual-website

One-page Astro site for [www.nwprtritual.com](https://www.nwprtritual.com).

A Halo × NWPRT co-branded retreat landing page — static build, deployed to Cloudflare Pages.

## Commands

```bash
npm run dev       # dev server at http://localhost:4321
npm run build     # static build → dist/
npm run preview   # preview the built site locally
```

Node ≥22.12 required.

## Brand reference

- `nwprt-brand-style-guide.md` — NWPRT visual identity (palette, type, voice rules)
- `CLAUDE.md` — project architecture + brand discipline notes for AI assistants
